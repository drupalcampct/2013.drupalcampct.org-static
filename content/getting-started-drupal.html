Getting Started with Drupal

Time: 
9:40-10:30 AM
Room number: 
220
Speaker: 
Stephen Cross
Beginner
Site Building

Getting starting with any new technology is a challenge. This session is for the new or experienced developer that is jumping into Drupal for the first time. We cover the steps for installing Drupal on a local machine, the components of a Drupal website and key concepts to get moving.

Agenda:

Installing Drupal
- thing you need
- basic steps
- tips

Drupal Components
- Functionality - Modules (Core and contributed)
- Visual Layout - Theme
- Distributions

Key Concepts
- Content Types, Nodes
- Blocks, Menus and Taxonomy

First Project
- What you get our of the box
- Choosing a theme or distribution
- Types features and modules to start with

Resources - where to get help.
